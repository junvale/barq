# BarQ
A barcode and QR code generator with web app and browser extension.

## Tech Stack
- Monorepo with pnpm
- React
- TypeScript
- Tailwind CSS
- Vite

## Requirements
- Extension
  1. Can switch to QR code using Tab key
- Web App
  1. Web App mode for batch generation
  2. Settings page: Basic settings, Barcode settings, QR code settings, Print settings
  3. Basic settings: Default display (Barcode or QR code)
